## 4. 프로젝트 기능

### 회원 기능(회원가입/로그인)

- 회원가입 시, 아이디, 닉네임, 비밀번호 정규표현식 필터링 적용 ( 회원정보 수정 시에도 적용)
- 회원정보 CRUD 기능
- 회원가입 시 약관동의 기능
- 유저 권한 변경 기능( 권한에 따라 사이트 관리 가능)

### 게시판 기능 (게시글, 댓글, 대댓글)

- 게시글 CRUD 기능
- 게시글, 유저 검색 기능
- 게시글 신고 기능 (super 계정이 처리)

### 관리자 기능 ( 사이트 관리 페이지)

- 권한을 받은 유저가 접근 가능, 권한은 admin이 부여
- 신고당한 게시글 검토 후 처리 가능
- 유저 삭제 기능
- 차량 정보 관리
- 고객정보관리
- 게시판 관리
  

### 예약 기능

- 원하는 차종
- 예약 기간
- 실시간 예약 기능 여부 확인
